I Numeri Complessi

Un viaggio nel piano complesso: dalla definizione alle applicazioni

Introduzione ai Numeri Complessi

Perché i numeri complessi?

I numeri reali non sono sufficienti per risolvere tutte le equazioni algebriche. Ad esempio, l'equazione:

x² + 1 = 0

non ha soluzioni nell'insieme dei numeri reali ℝ, poiché nessun numero reale elevato al quadrato può dare -1.

L'unità immaginaria

Introduciamo un nuovo numero, chiamato unità immaginaria, indicato con i, definito dalla proprietà:

i² = -1

Da questa proprietà fondamentale nascono tutti i numeri complessi.

📝 Appunti per gli studenti
  • L'unità immaginaria i è un concetto astratto ma fondamentale
  • Non cercare di "immaginare" i nel senso comune: è una definizione matematica
  • I numeri complessi estendono i numeri reali, non li sostituiscono
  • Applicazioni: elettrotecnica, fisica quantistica, ingegneria, elaborazione segnali

Storia e sviluppo

I numeri complessi furono sviluppati nel XVI secolo da matematici italiani come Cardano e Bombelli durante lo studio delle equazioni cubiche. Il termine "immaginario" fu coniato da Cartesio, inizialmente con connotazione negativa.

Definizione di Numero Complesso

Definizione formale

Un numero complesso z è un'espressione della forma:

z = a + bi

dove:

  • a è la parte reale di z, indicata con Re(z)
  • b è la parte immaginaria di z, indicata con Im(z)
  • i è l'unità immaginaria (i² = -1)
  • a, b ∈ ℝ (sono numeri reali)

L'insieme dei numeri complessi ℂ

L'insieme di tutti i numeri complessi si indica con ℂ:

ℂ = {a + bi | a, b ∈ ℝ}
Esempio 1: Numeri complessi
  • z₁ = 3 + 4i → Re(z₁) = 3, Im(z₁) = 4
  • z₂ = -2 + 5i → Re(z₂) = -2, Im(z₂) = 5
  • z₃ = 7 + 0i = 7 → numero reale puro
  • z₄ = 0 + 3i = 3i → numero immaginario puro

Complesso coniugato

Il complesso coniugato di z = a + bi è:

z̄ = a - bi

Si ottiene cambiando il segno della parte immaginaria.

📝 Appunti per gli studenti
  • Ogni numero reale è anche un numero complesso (con b = 0)
  • Quindi ℝ ⊂ ℂ (i reali sono un sottoinsieme dei complessi)
  • Due numeri complessi sono uguali se e solo se hanno stessa parte reale e stessa parte immaginaria
  • Il coniugato è importante per molte operazioni, specialmente la divisione

Il Piano di Gauss (Piano Complesso)

Rappresentazione geometrica

I numeri complessi possono essere rappresentati come punti o vettori nel piano di Gauss (o piano complesso o piano di Argand-Gauss):

  • Asse orizzontale (asse reale): rappresenta la parte reale
  • Asse verticale (asse immaginario): rappresenta la parte immaginaria
  • Il numero z = a + bi corrisponde al punto (a, b)
Piano di Gauss con esempi di numeri complessi

Modulo di un numero complesso

Il modulo (o valore assoluto) di z = a + bi è la distanza dall'origine:

|z| = √(a² + b²)

Argomento di un numero complesso

L'argomento di z è l'angolo θ che il vettore z forma con l'asse reale positivo:

arg(z) = θ = arctan(b/a)

L'argomento è definito a meno di multipli di 2π.

Esempio 2: Modulo e argomento

Sia z = 3 + 4i

  • Modulo: |z| = √(3² + 4²) = √(9 + 16) = √25 = 5
  • Argomento: θ = arctan(4/3) ≈ 0.927 radianti ≈ 53.13°
📝 Appunti per gli studenti
  • Il piano di Gauss permette di "vedere" i numeri complessi
  • Il modulo è sempre un numero reale non negativo
  • L'argomento si misura in radianti (o gradi)
  • Attenzione ai quadranti quando si calcola l'argomento!
  • Per z = 0, il modulo è 0 e l'argomento è indeterminato

Operazioni con i Numeri Complessi

1. Addizione

Dati z₁ = a + bi e z₂ = c + di:

z₁ + z₂ = (a + c) + (b + d)i

Si sommano separatamente le parti reali e le parti immaginarie.

Esempio 3: Addizione

(3 + 4i) + (2 - 5i) = (3 + 2) + (4 - 5)i = 5 - i

2. Sottrazione

z₁ - z₂ = (a - c) + (b - d)i
Esempio 4: Sottrazione

(3 + 4i) - (2 - 5i) = (3 - 2) + (4 - (-5))i = 1 + 9i

3. Moltiplicazione

Si applica la proprietà distributiva ricordando che i² = -1:

z₁ · z₂ = (a + bi)(c + di) = ac + adi + bci + bdi² = (ac - bd) + (ad + bc)i
Esempio 5: Moltiplicazione

(3 + 4i)(2 - 5i) = 6 - 15i + 8i - 20i² = 6 - 7i - 20(-1) = 6 - 7i + 20 = 26 - 7i

4. Divisione

Per dividere, si moltiplica numeratore e denominatore per il coniugato del denominatore:

z₁/z₂ = (a + bi)/(c + di) · (c - di)/(c - di) = [(ac + bd) + (bc - ad)i]/(c² + d²)
Esempio 6: Divisione

(3 + 4i)/(2 - i) = (3 + 4i)/(2 - i) · (2 + i)/(2 + i) = (6 + 3i + 8i + 4i²)/(4 - i²) = (6 + 11i - 4)/(4 + 1) = (2 + 11i)/5 = 2/5 + 11i/5

Visualizzazione geometrica delle operazioni
📝 Appunti per gli studenti
  • Addizione e sottrazione: interpretazione geometrica come somma/differenza di vettori
  • Moltiplicazione: i moduli si moltiplicano, gli argomenti si sommano
  • Divisione: i moduli si dividono, gli argomenti si sottraggono
  • Il coniugato è fondamentale per la divisione
  • Proprietà: commutativa, associativa, distributiva

Forma Trigonometrica (o Polare)

Definizione

Dato z = a + bi con modulo ρ = |z| e argomento θ = arg(z), la forma trigonometrica è:

z = ρ(cos θ + i sin θ)

dove:

  • ρ = √(a² + b²)
  • a = ρ cos θ
  • b = ρ sin θ

Conversione tra forme

z = a + bi
ρ = √(a² + b²)
θ = arctan(b/a) 		z = ρ(cos θ + i sin θ)
a = ρ cos θ
b = ρ sin θ
Esempio 7: Conversione in forma trigonometrica

Convertire z = 1 + i√3 in forma trigonometrica:

  • ρ = √(1² + (√3)²) = √(1 + 3) = √4 = 2
  • θ = arctan(√3/1) = arctan(√3) = π/3 (60°)
  • Quindi: z = 2(cos(π/3) + i sin(π/3))

Formula di De Moivre

Per calcolare potenze di numeri complessi in forma trigonometrica:

z^n = [ρ(cos θ + i sin θ)]^n = ρ^n(cos(nθ) + i sin(nθ))
Esempio 8: Applicazione Formula di De Moivre

Calcolare (1 + i)⁸:

  • Convertiamo: 1 + i = √2(cos(π/4) + i sin(π/4))
  • (1 + i)⁸ = (√2)⁸(cos(8π/4) + i sin(8π/4)) = 16(cos(2π) + i sin(2π)) = 16(1 + 0i) = 16
Rappresentazione forma trigonometrica
📝 Appunti per gli studenti
  • La forma trigonometrica è particolarmente utile per moltiplicazioni e potenze
  • In forma trigonometrica si vede chiaramente il modulo e l'argomento
  • La formula di De Moivre semplifica enormemente il calcolo di potenze
  • Attenzione al quadrante quando si calcola θ!

Forma Esponenziale (Formula di Eulero)

Formula di Eulero

Una delle formule più belle della matematica collega esponenziali e funzioni trigonometriche:

e^(iθ) = cos θ + i sin θ

Da questa formula, un numero complesso z = ρ(cos θ + i sin θ) può essere scritto come:

z = ρe^(iθ)

Caso speciale: Formula di Eulero con θ = π

Sostituendo θ = π nella formula di Eulero:

e^(iπ) = cos π + i sin π = -1 + 0i = -1

Da cui la famosa identità di Eulero:

e^(iπ) + 1 = 0

Questa formula collega i cinque numeri più importanti della matematica: e, i, π, 1, 0

Operazioni in forma esponenziale

Moltiplicazione:

z₁ · z₂ = ρ₁e^(iθ₁) · ρ₂e^(iθ₂) = ρ₁ρ₂e^(i(θ₁+θ₂))

Divisione:

z₁/z₂ = ρ₁e^(iθ₁) / ρ₂e^(iθ₂) = (ρ₁/ρ₂)e^(i(θ₁-θ₂))

Potenza:

z^n = (ρe^(iθ))^n = ρ^n e^(inθ)
Esempio 9: Operazioni in forma esponenziale

Siano z₁ = 2e^(iπ/6) e z₂ = 3e^(iπ/4)

  • z₁ · z₂ = 2·3 · e^(i(π/6 + π/4)) = 6e^(i5π/12)
  • z₁/z₂ = (2/3)e^(i(π/6 - π/4)) = (2/3)e^(-iπ/12)
  • z₁³ = 2³e^(i3π/6) = 8e^(iπ/2) = 8i

Radici n-esime

Le radici n-esime di z = ρe^(iθ) sono:

w_k = ρ^(1/n) · e^(i(θ + 2πk)/n), k = 0, 1, 2, ..., n-1
Esempio 10: Radici cubiche di -8

Troviamo le radici cubiche di -8:

  • -8 = 8e^(iπ) (forma esponenziale)
  • w₀ = 8^(1/3) · e^(iπ/3) = 2e^(iπ/3) = 2(1/2 + i√3/2) = 1 + i√3
  • w₁ = 2e^(i(π + 2π)/3) = 2e^(iπ) = -2
  • w₂ = 2e^(i(π + 4π)/3) = 2e^(i5π/3) = 2(1/2 - i√3/2) = 1 - i√3
Radici cubiche di -8 nel piano complesso
📝 Appunti per gli studenti
  • La forma esponenziale è la più compatta e elegante
  • È particolarmente utile in fisica e ingegneria
  • Le radici n-esime sono sempre n valori distinti, equidistanti su una circonferenza
  • La formula di Eulero è fondamentale in analisi complessa

Applicazioni Reali dei Numeri Complessi

1. Elettrotecnica - Impedenza

L'impedenza di un circuito AC è un numero complesso:

  • Z = R + iX, dove R è la resistenza e X la reattanza
  • Per un circuito con R = 30Ω e L = 0.1H a f = 50Hz:
  • X_L = 2πfL = 2π(50)(0.1) ≈ 31.4Ω
  • Z = 30 + 31.4i Ω
  • |Z| = √(30² + 31.4²) ≈ 43.4Ω
2. Fisica - Oscillazioni

Un'onda può essere rappresentata come:

  • ψ(x,t) = Ae^(i(kx - ωt)) dove A è l'ampiezza
  • La parte reale rappresenta l'oscillazione fisica
  • Facilita enormemente i calcoli con onde e oscillazioni
3. Elaborazione dei Segnali - Trasformata di Fourier

La trasformata di Fourier usa i numeri complessi:

  • F(ω) = ∫f(t)e^(-iωt)dt
  • Permette di analizzare segnali nel dominio della frequenza
  • Applicazioni: compressione audio/video, telecomunicazioni
4. Frattali - Insieme di Mandelbrot

L'insieme di Mandelbrot è definito tramite iterazione di numeri complessi:

  • z_{n+1} = z_n² + c
  • Dove c è un numero complesso costante
  • Un punto c appartiene all'insieme se la sequenza rimane limitata
